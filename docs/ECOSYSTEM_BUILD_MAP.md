# Ecosystem Build Map

This is the companion build map for the Sovereign Monad ecosystem.

Use it as the structural guide for where active work belongs.

## Active Domains

- `theo-techno-cosmo/` - philosophical and doctrinal material
- `gnostic-engine/` - volumetric 4D processing engine
- `monad-ecosystem/` - agents, control center, contracts, and system operations

## Supporting Areas

- `docs/` - canonical operating documents
- `docs/PROJECT_STATE.md` - resume checkpoint for the current project state
- `scripts/` - workspace orchestration and verification
- `archive/` - legacy, generated, and sandboxed material

## Reading Rule

If a file or folder conflicts with the MOF, the MOF wins.
