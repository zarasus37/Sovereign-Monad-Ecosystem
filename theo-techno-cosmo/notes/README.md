# notes/ — Reference Material & Working Documentation

The **notes/** directory contains reference material, working documentation, and supporting materials for Theo-Techno-Cosmo.

This is a **research and reference library**, not part of the three main layers (Wheel, Council, PLEX).

## What Goes Here?

- Study notes on primary and secondary sources
- Working papers and draft explorations
- Reference documentation compiled from external sources
- Annotations and cross-references between concepts
- Research in progress

## What Doesn't Go Here?

- Historical lineage documents → **THE COUNCILE/**
- Operative diagrams and visual maps → **Wheel/**
- Current operational doctrine and audit trails → **plex/**

## How to Use This Directory

This is a **research tool**, not a teaching tool. Use these materials to:

1. **Deepen understanding** of a concept encountered in Wheel, Council, or PLEX
2. **Research cross-references** between different traditions or frameworks
3. **Build your own synthesis** before contributing to formal sections
4. **Track working hypotheses** before they become PLEX research studies

## Organization

Materials are organized by topic or thinker where possible, but this is a working directory — organization may shift as research evolves.
