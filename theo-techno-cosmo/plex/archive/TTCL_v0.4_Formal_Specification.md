## 1. Foundation of the Language

TTCL is an intention-first, rootmology-grounded, modality-enforced language built on the LTP-ML substrate (Llull–Trithemius–Peirce Machine Language).

**Core Unit:** The **Sign**  
- id: immutable unique identifier  
- label: human-readable text  
- type_signature: base + modal type  
- modality: Icon | Index | Symbol | Hybrid  
- payload: implementation or description  
- provenance: token tracing wheel slot + key + constitution version  
- svg_anchor: reference to visual wheel element  

**Canonical Wheels (loaded from provided images):**  
- Llull’s Tabula Generalis & four figures (combinatorial alphabet)  
- Trithemius Steganographia circle (keyed rotations/encodings)  
- Teologia wheel (theological attributes)  
- Demonstrative Terra wheel (elemental emergence)  
- Radiant A-centered wheel (sovereign unity)  

**Core Law:**  
**Intention (tripartite-saturated key) → applied definitive meaning through dynamic keyed rootmology association → governed by enforceable semiotic contracts (Icon/Index/Symbol) → compiled through modality-aware Sign Graph IR → executed on provenance-aware VM — all self-evaluated against the versioned LOGOC Constitution.**

No prefixes/suffixes. Meaning emerges through compression via the wheels.
