# Theo-Techno-Cosmo

## Purpose

Theo-Techno-Cosmo is the philosophical and doctrinal pillar of the Sovereign Monad ecosystem. It holds the TTC / LOGOC framework, the layered emanation model, and the historical lineages that inform the system's operating assumptions.

## Contents

- `Wheel/` - contemplative maps and operative diagrams
- `THE COUNCILE/` - historical sources and philosophical lineages
- `plex/` - live manifestation, review, research, and archive material for TTC
- `notes/` - supporting notes and working references

## Run
