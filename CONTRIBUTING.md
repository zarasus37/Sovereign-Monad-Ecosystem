# Contributing

This repository is the unified workspace for the Sovereign Monad ecosystem.

## Open First

Before making changes, read:

1. `README.md`
2. `docs/SOVEREIGN_MONAD_ECOSYSTEM_MASTER_OPERATING_FILE_v2.4.0.md`
3. `docs/PROJECT_STATE.md`
4. `docs/REPO_STRUCTURE_MAP.md`

## Working Rules

- Keep active work in the three pillars:
  - `theo-techno-cosmo/`
  - `gnostic-engine/`
  - `monad-ecosystem/`
- Keep archive material under `archive/`.
- Prefer repo-relative paths over absolute drive paths.
- Do not introduce new top-level folders unless they are part of the root contract.
- Run `pnpm check:layout` after structural changes.

## Archive Rule

- Archive material must describe itself as archive material.
- Do not copy old workspace names back into active docs or scripts.
